# Gerador de CPF com Integração Gráfica em Java

Este projeto em Java é um **Gerador de CPF** com interface gráfica, utilizando **JFrame** e **Programação Orientada a Objetos (POO)**. Ele gera números de CPF válidos e os exibe no formato **xxx.xxx.xxx-xx**, garantindo que o CPF gerado siga as regras da Receita Federal.

## Funcionalidades

- Geração automática de CPFs válidos.
- Interface gráfica simples e intuitiva com **JFrame**.
- Validação de CPFs gerados conforme as regras da Receita Federal.
- Exibição do CPF no formato **xxx.xxx.xxx-xx**.

## Tecnologias Utilizadas

- **Java**: Linguagem de programação principal.
- **JFrame**: Biblioteca para construção da interface gráfica.
- **Swing**: Framework para construção da interface com o usuário.
- **Programação Orientada a Objetos (POO)**: Organização do código em classes e objetos.
